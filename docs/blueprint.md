# **App Name**: AI Learning Arcade

## Core Features:

- Gamified Learning Modules: Interactive lessons for basic, intermediate, and advanced AI concepts presented as mini-games.
- AI Tool Recommendations: Contextual recommendations for AI tools (RunwayML, Google Teachable Machine, HuggingFace Spaces, Artbreeder) based on the user's current learning module and progress. LLM will use its reasoning as a tool to decide when to recommend each tool
- Progress Tracking: Visual representation of the user's progress through the AI learning path.
- Resource Library: Curated list of external resources, including links to RunwayML (https://runwayml.com), Google Teachable Machine (https://teachablemachine.withgoogle.com), OpenProcessing (https://www.openprocessing.org), HuggingFace Spaces (https://huggingface.co/spaces), Artbreeder (https://www.artbreeder.com), Wikimedia Commons (https://commons.wikimedia.org), Biblioteca Digital Mundial (UNESCO) (https://www.wdl.org/es/), Museo de Arte Moderno de Medellín (MAMM) (https://www.elmamm.org), and Banco de la República - Biblioteca Virtual (https://www.banrepcultural.org/biblioteca-virtual).
- Community Showcase: A space where users can share their AI projects and creations made using the recommended tools.

## Style Guidelines:

- Primary color: Saturated violet (#9400D3) to reflect the creative and innovative nature of AI learning.
- Background color: Light gray (#F0F0F0) for a clean and modern look.
- Accent color: Bright yellow (#FFD700) to highlight interactive elements and calls to action, creating an energetic feel.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-forward feel.
- Body font: 'Inter' (sans-serif) for readability and clean presentation of content.
- Use playful, custom icons for each AI concept and tool category.
- Subtle animations and transitions to create an engaging and dynamic user experience, particularly when navigating between learning modules.
